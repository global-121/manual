---
title: Suspendre les paiements
hide:
  - toc
---

Tant qu'une inscription a le statut « Actif », elle reçoit automatiquement un nouveau paiement chaque semaine. Si nécessaire, vous pouvez suspendre temporairement les paiements :

1. Trouvez l'inscription (les inscriptions)
2. Sélectionnez la (les) personne(s) pour lesquelles vous souhaitez suspendre l'assistance
3. Cliquez sur Suspendre
4. Dans la fenêtre contextuelle, vérifiez « Envoyer un message à l'inscription(aux inscriptions) »
5. Un message automatique sera envoyé dans la langue préférée
6. Cliquez sur approuver

Vous pouvez également cliquer avec le bouton droit sur l'inscription dans la liste et sélectionner Suspendre
