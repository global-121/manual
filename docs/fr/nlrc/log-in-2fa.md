title: Se connecter avec 2FA
hide:

---

1. Ouvrez un navigateur Internet (Chrome, Firefox, Safari ou Microsoft Edge).
    - Remarque : 121 ne fonctionne pas dans Internet Explorer.
2. Allez à [le portail 121](https://new.portal.nlrc.121.global){:target="_blank"}.
3. Entrez votre adresse e-mail et cliquez sur *Se connecter*. </br>
   ![connexion1](../../assets/img/nlrc/nl/2falogin1.png){: width="30%" }
4. Cliquez sur « Envoyer la notification », un code apparaîtra à l'écran </br>
   ![connexion2](../../assets/img/nlrc/nl/2falogin2.png){: width="30%" }
   ![connexion3](../../assets/img/nlrc/nl/2falogin3.png){: width="30%" }
5. Si vous avez configuré ceci, vous recevrez une notification sur votre téléphone. Sinon, ouvrez votre application d'authentification sur votre téléphone.
6. Entrez le code de l'écran sur votre téléphone pour vous connecter. </br>
   ![connexion4](../../assets/img/nlrc/nl/2falogin4.png){: width="30%" }
7. Vous êtes connecté!
8. Cliquez sur un projet pour continuer.

Vous verrez un aperçu avec le nombre total d'enregistrements, les demandeurs d'aide actifs et le montant total qui a été versé. Cliquez sur le nom d'un projet pour continuer.
