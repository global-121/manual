---
title: Terminer l'assistance
hide:
  - toc
---

Si une personne n'a plus besoin d'assistance, vous pouvez arrêter l'assistance. Pour ce faire, contactez votre personne de contact à la Croix-Rouge.

Sinon:

1. Trouvez le bénéficiaire dans le tableau
2. Sélectionnez la (les) personne(s)
3. Cliquez sur Décliner
4. Cliquez sur approuver

L'assistance s'arrête immédiatement.

Vous pouvez également cliquer avec le bouton droit sur le bénéficiaire et choisir Décliner
