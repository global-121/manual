---
title: Modifier les inscriptions
hide:
  - toc
---

1. Trouvez l'inscription dans la liste.
2. Cliquez sur le numéro d'enregistrement ou le nom pour ouvrir le profil.
   En haut, vous verrez les informations clés; en bas, le journal des activités.
3. Cliquez sur Informations personnelles pour afficher les détails.
4. Cliquez sur Modifier les informations sur la droite et apportez les modifications.
5. Cliquez sur Enregistrer.
6. Entrez la raison du changement et cliquez à nouveau sur Enregistrer.

!!! info
    Le nombre de paiements hebdomadaires ne peut pas être modifié dans 121. Pour une assistance supplémentaire, contactez la Croix-Rouge.
