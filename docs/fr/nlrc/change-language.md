Vous pouvez modifier la langue à deux moments :

## Après vous être connecté

1. Cliquez sur le bouton de menu (:material-menu:) en haut à gauche de la page.
2. Choisissez une langue dans le menu en bas.

![Sélecteur de langue](../../assets/img/ChangeLanguage.png)

## Avant de vous connecter

1. Choisissez une langue dans le menu en bas de la page de connexion.

![Page de connexion](../../assets/img/LoginPage.png)
