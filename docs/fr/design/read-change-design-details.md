!!! info "Cette section explique"
    Comment lire et modifier les métriques dans la conception du programme.

### Lire les métriques des détails du programme

Les **Métriques des détails du programme** traduisent vos exigences de programme. Les détails fournissent l'information principale sur la durée et l'emplacement de votre programme, le budget, les personnes touchées, la prestation de l'aide monétaire et les fréquences de distribution. Chacun de vos programmes actifs peut afficher différentes métriques présentées dans le tableau ci-dessous.

![Détails du programme](https://raw.githubusercontent.com/global-121/121-platform/main/e2e/tests/__screenshots__/UserManualScreenshots/userManualScreenshots.spec.ts/ProgramDetails.png)

| Type                                 | Description                                                                 |
| :----------------------------------- | :-------------------------------------------------------------------------- |
| **Titre Portal**                     | Titre du programme. Par exemple, "Aide en espèces polyvalente" ou "Réponse conjointe" |
| **Date de début et de fin**               | Information sur le début et la fin de votre programme d'aide en espèces                |
| **Emplacement**                         | Pays ou région dans lequel le programme se déroule​                          |
| **Fournisseur de services financiers (FSP)** | Nombre de FSP intégrés ou enregistrés dans votre instance                     |
| **Fréquence de distribution**           | Fréquence d'émission des paiements (quotidienne, hebdomadaire, mensuelle, trimestrielle, annuelle)   |
| **Durée de distribution**            | Nombre de cycles de paiement qui seront émis pendant la durée du programme      |
| **Valeur de transfert**                   | Nombre de FSP intégrés ou enregistrés dans votre instance                     |
| **Budget total du projet**             | Nombre de FSP intégrés ou enregistrés dans votre instance                     |
| **PA ciblées**                      | Nombre total de bénéficiaires ciblés par ce programme             |
| **Dernière mise à jour + Date**              | Date et heure de la dernière mise à jour du tableau                              |

### Modifier les métriques des détails du programme

Lors de la configuration de votre instance, notre équipe vous demandera de partager vos **détails du programme**. Ces métriques peuvent être ajustées en fonction de vos exigences de programme et pendant la durée du programme. Par exemple, les dates, la fréquence et la durée peuvent parfois changer pendant que votre programme est en cours.

!!! Warning "Mise à jour de vos métriques"
    Actuellement, il n'est pas possible d'effectuer des actions dans la conception du programme ou de modifier vous-même ces métriques. Veuillez [contacter l'équipe d'assistance 121](mailto:support@121.global) pour vous aider avec ceci.
