La **page Suivi** peut comporter un tableau de bord Power BI, soit la version de modèle 121, soit une version personnalisée, si vous l'avez demandée.

!!! Important "Qui peut effectuer des actions sur cette page?"
    Tous les utilisateurs. [En savoir plus sur les rôles utilisateur.](../users/users-roles-page.md)

## Accès à la section PowerBI

- Allez au **Programme**.
- Accédez à la **page Suivi**.
- Sélectionnez l'onglet **PowerBI**.

![Page Suivi](../assets/img/MonitoringPage.png)

## Comment afficher le tableau de bord PowerBI

Un **modèle de tableau de bord Power BI standard** a été ajouté à toutes les instances 121 par défaut. Ce modèle comprend des indicateurs dérivés des programmes gérés au sein de la plateforme et est basé uniquement sur les données disponibles dans la plateforme, telles que les fréquences de paiement, les débursements, l'état des paiements, le nombre d'enregistrements, le nombre d'inclusions, les doublons et les métriques similaires.

Les **tableaux de bord Power BI personnalisés** peuvent également être intégrés. Pour ce faire, **veuillez partager le lien du rapport Power BI** avec notre équipe afin qu'il puisse être affiché dans votre instance. Cela vous permet d'accéder à un aperçu avec des informations plus détaillées adaptées à vos besoins, par exemple par emplacement, groupe d'âge ou d'autres critères.

*Le tableau de bord est actualisé toutes les heures.*

![Tableau de bord PowerBI](../assets/img/PowerBI.png)

!!! Question "Construire votre propre tableau de bord"
    Si vous souhaitez créer votre propre tableau de bord mais que vous avez une expérience limitée avec Power BI, vous pouvez planifier un appel avec notre équipe pour discuter de vos besoins spécifiques et explorer les options disponibles.

    Dans la plupart des cas, des connaissances avancées en Excel et une compréhension de base de Power BI sont requises pour développer un tableau de bord personnalisé.
