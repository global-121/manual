**Kobo Toolbox est directement intégré à la plateforme 121**. Les enregistrements sont automatiquement ajoutés à la plateforme 121 en temps réel. Pour configurer cette intégration, notre équipe d'assistance doit accéder à votre formulaire Kobo pour créer le lien entre Kobo et la plateforme 121. Une fois cette intégration finalisée, vous pouvez révoquer l'accès de notre équipe d'assistance si vous préférez.

*Préférez-vous importer votre liste via Excel? Suivez les instructions sur [comment importer les enregistrements.](../registration/registration-import-list.md)*

!!! Important "Qui peut effectuer des actions sur cette page ?"
    L'administrateur du programme, l'agent d'assistance et le gestionnaire d'assistance peuvent effectuer des actions.
    Des utilisateurs supplémentaires peuvent uniquement avoir des permissions d'affichage sur cette page. [En savoir plus sur les utilisateurs et les rôles](../users/users-roles-page.md)

## Pour configurer l'intégration Kobo dans votre programme 121

À moins que vous n'importiez manuellement les enregistrements avec un fichier Excel .CSV, les enregistrements faits avec KOBO seront automatiquement importés dans votre programme 121, dans la page **Enregistrements**.

- Créez votre formulaire Kobo sur l'outil KOBO;
- Partagez ce formulaire Kobo avec le compte Kobo (demandez à notre équipe d'assistance);
- Informez l'équipe 121 que vous avez partagé le formulaire.

Notre équipe d'assistance 121 liera le formulaire kobo au programme approprié dans la plateforme 121 et vous informera quand il sera prêt. À moins que des modifications supplémentaires ne soient requises, le mappage de votre formulaire à la plateforme 121 sera effectué dans les 1 à 2 jours ouvrables suivant la réception de la demande.

!!! info "KOBO pour les Sociétés nationales de la Croix-Rouge et du Croissant-Rouge"
    La plateforme 121 est un produit de la Croix-Rouge néerlandaise. Les Sociétés nationales de la Croix-Rouge et du Croissant-Rouge peuvent créer leur formulaire kobo via [le serveur IFRC](https://kobo.ifrc.org/). Une fois créé, vous pouvez partager ce formulaire Kobo avec le compte `nlrc121platform`.

### Afficher une liste d'enregistrement KOBO

- Allez à votre **programme;**
- Entrez la page **Enregistrements**;
- Vous verrez la liste mise à jour des enregistrements chaque fois qu'un nouveau formulaire est complété et envoyé (lorsque connecté à Internet).

Notez que la boîte à outils KOBO peut être utilisée hors ligne, sur le terrain. Néanmoins, la liste d'enregistrement ne peut être importée dans la plateforme 121 que lorsqu'une connexion Internet est établie.

### Liste vide

Voyez-vous une liste vide lorsque vous êtes sur la page d'enregistrements? Le tableau d'enregistrement peut être vide pour plusieurs raisons:

- Aucun nouvel enregistrement n'a été téléchargé depuis le dernier tour de validation. Le tableau affichera « Aucune donnée »;
- Aucun téléchargement automatique n'a été effectué à partir du formulaire KOBO ou d'une application d'enregistrement alternative encore. Ceci est généralement causé par un manque de connexion Internet. Pour commencer à importer de nouveaux enregistrements, connectez l'appareil avec le formulaire KOBO ouvert à un réseau Internet;
- Des filtres sont appliqués à la liste. Assurez-vous que les filtres sont clairs, en haut du tableau.

!!! warning "Page vierge"
    Si la page est complètement vierge et n'affiche pas le tableau d'enregistrement, il peut y avoir un problème technique avec le lien reliant votre formulaire KOBO.
    Veuillez contacter notre [équipe d'assistance](mailto:support@121.global) pour aider à résoudre le problème.

Les entrées en double peuvent se produire lors des enregistrements sur le terrain. Le dédoublement peut être effectué dans la plateforme 121. Vous pouvez suivre les étapes sur [cette page.](./registration-deduplication.md)

## Données envoyées de KOBO à la plateforme 121

Notre équipe liera toutes les questions ajoutées à votre KOBO à la plateforme 121, les données seront affichées dans le profil d'enregistrement. Cela peut inclure des questions - réponses, chiffres, nombres, etc.

Il n'y a pas besoin de suivre des modèles car ceci est adaptable dans la plateforme 121. Cependant, vous pouvez considérer si toutes les données de votre formulaire KOBO sont pertinentes pour l'évaluation d'inclusion de vos enregistrements dans le programme. Notre équipe de gestion de l'aide alimentaire et financière peut conseiller certaines modifications, si nécessaire, pour optimiser les données rapportées dans la plateforme 121, faciliter le calcul du score d'inclusion et les rapports de programme.

!!! Info "Modification des données"
    Veuillez noter que la connexion entre KOBO et la plateforme 121 est un téléchargement unidirectionnel. Cela signifie que les données ne sont pas synchronisées entre les deux systèmes. Toute modification effectuée dans KOBO après le téléchargement des données dans la plateforme 121 ne sera pas répercutée dans la plateforme. Vous devez effectuer les modifications directement dans la plateforme 121.

    Par exemple, si l'enregistrement A est enregistré dans KOBO et que vous téléchargez les données dans la plateforme 121 à la fin de la journée, vous pourriez plus tard remarquer que la question 4 n'a pas été répondue par l'enregistrement A. Dans ce cas, vous devrez trouver le profil d'enregistrement A dans la plateforme 121 pour effectuer les modifications nécessaires. Les modifications apportées au formulaire d'enregistrement KOBO A après le téléchargement initial ne seront pas mises à jour dans la plateforme 121.
