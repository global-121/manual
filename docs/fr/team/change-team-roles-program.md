Dans la page **Paramètres** de votre programme, sous l'onglet **Équipe du programme**, vous pouvez voir la liste des utilisateurs actuellement ajoutés à votre programme, avec leurs rôles assignés. Si vous avez la permission, vous pouvez **modifier, ajouter ou supprimer un rôle assigné à un utilisateur**.

!!! Important "Qui peut effectuer des actions sur cette page ?"
    L'administrateur du programme et le gestionnaire d'assistance peuvent effectuer des actions sur cette page.
    Des utilisateurs supplémentaires peuvent uniquement avoir des permissions d'affichage sur cette page. [En savoir plus sur les utilisateurs et les rôles](../users/users-roles-page.md)

### Modifier les rôles assignés et les permissions

- Entrez le programme souhaité et **allez à :material-cog-outline: Paramètres**,
- Sélectionnez l'onglet **Équipe du programme**,
- Sur le côté droit de votre écran, cliquez sur **:octicons-pencil-24: le crayon**

![Ajouter un membre de l'équipe](../assets/img/SettingsProjectTeam.png)

- Cliquez sur les 3 points **:material-dots-horizontal:** à la fin de la ligne utilisateur,
- Sélectionnez **Modifier**

![Bouton Modifier les rôles](../assets/img/EditRoles.png)

- Dans le champ **Assigner les rôles**, **Sélectionnez ou désélectionnez** les rôles que vous souhaitez assigner à l'utilisateur.
- Cliquez sur **Enregistrer les modifications** pour confirmer.

![Fenêtre pop-up Rôles utilisateur](../assets/img/AddUserWindow.png)

!!! question "Ajouter ou supprimer des utilisateurs"
    Vous pouvez suivre les instructions pour [ajouter des utilisateurs à un programme](../team/add-team-members.md) ou pour [révoquer l'accès de l'utilisateur d'un programme.](../team/remove-team-members-program.md)
