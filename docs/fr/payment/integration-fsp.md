    L'intégration existante avec PSF local et demande de nouvelle intégration.

| Pays                            | Nom du PSF                    | Méthodes de livraison activées    |
| :------------------------------ | :---------------------------- | :-------------------------------- |
| Kenya                           | MPESA Safaricom               | Monnaie mobile                    |
| Éthiopie                        | Banque commerciale d'Éthiopie | Virement bancaire                 |

| Afrique du Sud                  | MTN                           | Portail PSF                       |

!!! info "Agrégateur"
    En cours d'intégration avec un agrégateur africain pour couvrir jusqu'à X pays, permettant d'activer instantanément les paiements avec PSF locaux en pays. Processus plus rapide.


### Méthodes de livraison

| Catégorie | Description |
| ---------------- | ----------- |
| Portail PSF | Export manuel des instructions de paiement de 121 pour téléchargement sur le portail PSF. Réconciliation manuelle de la confirmation de paiement dans la plateforme 121 pour mettre à jour le statut de paiement. |
| Virement bancaire | Les virements bancaires directs émis avec la plateforme 121 aux enregistrements (PSF intégré) |

### Demande de nouvelle intégration

Exigences :

- API
- Méthodes de paiement prises en charge
- Contrats avec la Société nationale RCRC et/ou les ONG internationales

Calendrier


!!! info "Exporter les rapports de paiement"
    Exportez les fichiers à partager avec les donateurs ou l'équipe financière une fois le paiement terminé.
