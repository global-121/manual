Si vous avez perdu ou oublié votre mot de passe, vous pouvez contacter votre gestionnaire d'utilisateurs 121. Il peut réinitialiser votre mot de passe via la page Utilisateurs comme suit.

!!! Important "Qui peut effectuer des actions sur cette page ?"
    Le gestionnaire d'utilisateurs peut consulter et effectuer des actions sur cette page.

---

### Réinitialiser le mot de passe d'un utilisateur

Le gestionnaire d'utilisateurs peut réinitialiser directement les mots de passe des utilisateurs à partir de la **page Utilisateurs**.

- À la fin de la ligne de l'utilisateur, cliquez sur les **:material-dots-horizontal: trois points**,
- Sélectionnez **Réinitialiser le mot de passe**,
- Confirmez l'action.

![Réinitialiser le mot de passe utilisateur](../assets/img/ResetPasswordUser.png)

**Un e-mail sera automatiquement envoyé à l'utilisateur pour qu'il puisse réinitialiser son mot de passe.** Une fois que les utilisateurs se sont connectés avec leurs nouveaux mots de passe, nous leur recommandons vivement de modifier et de personnaliser leur nouveau mot de passe pour plus de sécurité.

---

Si vous ne pouvez toujours pas vous connecter, veuillez contacter notre équipe d'assistance via <support@121.global>

!!! question "Modifier votre mot de passe actuel"
    Vous pouvez modifier votre mot de passe actuel dans votre profil utilisateur 121.  
    Suivez les instructions : [Modifier le mot de passe actuel](./change-current-password.md).
